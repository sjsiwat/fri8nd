# วิสัยทัศน์ (Vision)

fri8nd AI คือผู้ช่วย AI แบบ Local-first ที่ช่วยจัดระเบียบ เข้าใจ และบริหารจัดการองค์ความรู้ดิจิทัลของผู้ใช้โดยอัตโนมัติ

fri8nd ไม่ได้เป็นเพียงโปรแกรมจัดการไฟล์ แต่เข้าใจว่าไฟล์แต่ละไฟล์คืออะไร ควรอยู่ที่ไหน และมีความสัมพันธ์กับข้อมูลอื่นอย่างไร

แนวคิดของ fri8nd คือ

> Finder + Obsidian + Raycast + AI + Automation

fri8nd ทำงานอยู่เบื้องหลังตลอดเวลา เหมือน Dropbox หรือ Google Drive โดยไม่รบกวนผู้ใช้

---

# พันธกิจ (Mission)

ลดความรกของข้อมูลดิจิทัล

เปลี่ยนไฟล์ที่กระจัดกระจายให้กลายเป็นองค์ความรู้ที่ค้นหาและใช้งานได้

ช่วยให้ผู้ใช้ใช้เวลาสร้างและเรียนรู้ มากกว่าการเสียเวลาจัดไฟล์

---

# ปรัชญาของผลิตภัณฑ์

- Local First
- Privacy First

fri8nd จะไม่บังคับให้ผู้ใช้เก็บข้อมูลไว้บน Cloud ของบริษัท

# ปัญหาที่ต้องการแก้

ทุกวันนี้โฟลเดอร์ในคอมพิวเตอร์เต็มไปด้วยไฟล์ที่ไม่มีระเบียบ

- Downloads รก
- PDF จำนวนมาก
- รูปภาพตั้งชื่อไม่สื่อความหมาย
- เอกสารกระจัดกระจาย
- จำได้ว่าเคยอ่าน แต่จำชื่อไฟล์ไม่ได้

File Manager ในปัจจุบันค้นหาได้จาก "ชื่อไฟล์"

แต่ fri8nd ค้นหาได้จาก "ความหมายของข้อมูล"

# คุณค่าที่มอบให้

fri8nd ไม่ใช่ File Manager

fri8nd คือ AI Knowledge Assistant

ผู้ใช้ไม่จำเป็นต้องถามว่า

"ไฟล์นี้เก็บไว้ที่ไหน"

แต่สามารถถามว่า

"โน้ต MongoDB ที่อ่านเมื่อสัปดาห์ก่อนอยู่ไหน"

fri8nd จะเข้าใจความหมายของคำถาม

# เสาหลักของผลิตภัณฑ์

## 1. AI File Organizer

จัดการไฟล์อัตโนมัติ

- ตั้งชื่อใหม่
- สร้างโฟลเดอร์
- ย้ายไฟล์
- รวมโฟลเดอร์
- ตรวจจับไฟล์ซ้ำ

---

## 2. AI Knowledge Engine

เข้าใจข้อมูลจาก

- PDF
- รูปภาพ
- Markdown
- Office Documents
- Source Code

วิเคราะห์

- หัวข้อ
- Tags
- Metadata
- ความสัมพันธ์ของข้อมูล

---

## 3. ค้นหาด้วยภาษาธรรมชาติ

ตัวอย่าง

- หาโน้ต MongoDB ที่อ่านเมื่ออาทิตย์ก่อน
- แสดงไฟล์ทั้งหมดเกี่ยวกับ Docker
- ใบเสร็จของ Apple อยู่ที่ไหน
- เคยอ่าน OAuth จากเอกสารไหน

---

## 4. Background Agent

ทำงานเบื้องหลังตลอดเวลา

เริ่มทำงานอัตโนมัติเมื่อเปิดเครื่อง

เฝ้าดูโฟลเดอร์ที่กำหนด

วิเคราะห์ไฟล์ใหม่โดยอัตโนมัติ

ผู้ใช้แทบไม่ต้องกดอะไรเลย

---

## 5. Personal Knowledge Graph

สร้างความเชื่อมโยงระหว่าง

- บุคคล
- โปรเจกต์
- เทคโนโลยี
- บริษัท
- หัวข้อ
- ประวัติการเรียนรู้

โดยอัตโนมัติ

# วิสัยทัศน์ระยะยาว

fri8nd ไม่ต้องการเป็นเพียงแอปจดโน้ต

ไม่ต้องการเป็น Cloud Drive

ไม่ต้องการเป็น File Manager

fri8nd ต้องการเป็น

> "ระบบปฏิบัติการสำหรับการจัดการองค์ความรู้ส่วนบุคคล ที่มี AI เป็นผู้ช่วย"

เป้าหมายคือให้ผู้ใช้สามารถโฟกัสกับการเรียนรู้ การทำงาน และการสร้างสรรค์ โดยปล่อยให้ fri8nd จัดการความยุ่งเหยิงของข้อมูลแทน
